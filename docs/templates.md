# Medan Template Rasmi

Ringkasan ini diekstrak daripada dokumen contoh yang diberikan.

## Borang Penceramah

- Nama penuh
- No. Kad Pengenalan
- No. telefon
- Warganegara
- Alamat tempat tinggal
- Nama organisasi dan alamat bertugas
- Kelulusan akademik
- Kelulusan profesional
- Pentauliahan / sijil
- Nama pertubuhan / syarikat jika berkaitan
- Alamat rasmi pertubuhan / syarikat
- No. sijil pendaftaran
- Bayaran jika berkenaan
- Tarikh dan hari program
- Tajuk ceramah
- Butiran ringkas ceramah
- Tandatangan, nama, cap dan tarikh

## Surat Iringan Sekolah Kepada PPD

Struktur contoh:

- Kepala surat sekolah
- Rujukan tuan
- Rujukan kami
- Tarikh
- Alamat Pegawai Pendidikan Daerah Klang
- Perkara: jemputan penceramah sebagai bengkel / program
- Senarai dokumen disertakan:
  - Salinan surat jemputan penceramah
  - Salinan borang penceramah
  - Salinan kad pengenalan penceramah
- Slogan rasmi
- Tandatangan dan nama pengetua / guru besar

## Surat Jemputan Sekolah / PIBG Kepada Penceramah

Struktur contoh:

- Kepala surat sekolah / PIBG
- Rujukan kami
- Tarikh
- Nama dan alamat penerima / penceramah
- Perkara: jemputan mengendalikan bengkel / ceramah
- Ringkasan tujuan program
- Ketetapan program:
  - Tarikh
  - Masa
  - Tempat
  - Sasaran peserta
- Pegawai untuk dihubungi
- Slogan rasmi
- Tandatangan dan nama pengetua / guru besar
